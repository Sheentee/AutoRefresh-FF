# AutoRefresh

A clean, modern, and efficient browser extension that automatically refreshes pages after a customizable interval.

![AutoRefresh Interface](icons/icon128.png)

## Features

- **Customizable Intervals**: Set your preferred refresh rate in seconds.
- **Real-time Countdown**: Visual countdown showing exactly when the next refresh will occur.
- **Refresh Counter**: Keep track of how many times the page has been refreshed.
- **Easy Control**: Simple Start/Stop functionality with a single click.
- **Modern UI**: A premium, glassmorphism-inspired design that matches modern browser aesthetics.
- **Persistent State**: Remembers your settings for each tab.

## Installation

### For Developers (Manual Installation)

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Sheentee/AutoRefresh.git
    ```
2.  **Open Chrome Extensions**:
    Navigate to `chrome://extensions/` in your browser.
3.  **Enable Developer Mode**:
    Toggle the **Developer mode** switch in the top right corner.
4.  **Load Unpacked**:
    Click the **Load unpacked** button and select the `AutoRefresh` folder from your local machine.

## 🛠️ Usage

1.  Click on the AutoRefresh icon in your extension toolbar.
2.  Enter the desired refresh interval in seconds (e.g., `30`).
3.  Click **Start Refreshing**.
4.  The extension will now automatically refresh the active tab at the specified interval.
5.  To stop, simply open the popup again and click **Stop**.
